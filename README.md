# 🏋️ Workout Tracker — План тренировок в зале

Веб-приложение для записи и отслеживания результатов силовых тренировок.  
Фиксируйте дни тренировок, добавляйте упражнения с подходами, указывайте вес и время отдыха.  
Доступно с любого устройства через интернет благодаря встроенному туннелю ngrok.

## ✨ Возможности

- 📅 Создание и редактирование тренировочных дней с названием (например, «Спина и бицепс»).
- 🏋️ Добавление упражнений с произвольным количеством подходов.
- ⚖️ Учёт рабочего веса в каждом подходе (опционально).
- ⏱️ Время отдыха между подходами (опционально).
- 📊 Удобный просмотр всех тренировок с раскрывающимися карточками.
- 💾 Два режима хранения: JSON-файл (для простоты) или PostgreSQL (для надёжности и масштабирования).
- 🐳 Полная контейнеризация — запуск одной командой без установки Python и СУБД.
- 🌍 Автоматический публичный URL через ngrok — открывается на телефоне, планшете, любом устройстве.
